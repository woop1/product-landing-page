## 🎨 CSS Variables

## 🧾 Validación del formulario

| Campo | Atributos | Validación |
|------|----------|------------|
| Nombre | required, minlength="3" | mínimo 3 caracteres |
| Email | required, type="email" | formato válido |
| Teléfono | required, pattern="[0-9]{9}" | 9 dígitos |
| Motivo | required | selección obligatoria |
| Mensaje | required, minlength="10" | mínimo 10 caracteres |
| Checkbox | required | aceptación obligatoria |
| Fecha contacto | type="date" | selector de fecha |
